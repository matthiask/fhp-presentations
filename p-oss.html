Was ist Open Source Software?

Definition

  • Freie Weitergabe
    • Lizenzgebühr
    • Selbstkostenaspekt
  • Verfügbarer Quellcode
  • Abgeleitete Arbeiten
    • Explizit erlaubt
    • Schutz des Produkte- und Produzentennamens möglich
  • Keine Diskriminierungen von Personen oder Gruppen, keine Nutzungseinschränkung
  • …

Wichtige Lizenmodelle

  • Copyleft (GPL)
    • Abgeleitete Arbeiten müssen zwingend unter der GPL veröffentlicht werden.
    • Keine Weitergabe von Programmen ohne Quellcode.
  • BSD / MIT / Apache Lizenz
    • Haftungsausschluss
    • Teilweise Schutz von Namensrechten
    • Schutz vor Patentklagen (Apache License 2.0)

Bedeutung, Kosten und Nutzen von OSS

Ims Internet

  • «Das Internet» funktioniert nicht ohne OSS
  • Z.B. Webserver
    • Apache 64%
    • Microsoft IIS: 14%
    • nginx: 9%

Bei FEINHEIT

  • Python, Django, jQuery, Firefox, Chromium, Webkit (Safari, iOS Browser, Android Browser), Twitter Bootstrap, Apache, nginx, MySQL, memcached, redis, Postgres, Linux, Cyberduck, Asterisk
  • FeinCMS, Plata, Towel, Elephantblog, PDFDocument
  • Bald auch Teile von Keetab?

Was uns Open Source bringt und kostet

  • Positiv
    • Weniger Lizenzkosten
    • Entwicklercommunity welche Verbesserungen am FeinCMS beisteuert
    • Respekt
    • …
  • Negativ
    • Betreuungsaufwand für Dritte
    • Dokumentationsaufwand (naja)
    • Gesamt sicher mehrere hundert Stunden bisher
    • …

Nicht ausgeschöpftes Potential

Potential :-(

  • Wenig gutes Personal aufgrund Entwicklungsarbeit akquiriert
  • Wenig Agenturaufträge
    • z.B. Anfragen nach NASA Artikel in Sonntagszeitung
    • Glaubwürdigkeit, aber erst wenn Fuss in Türe ist
  • Gegenbeispiel: divio AG
    • django SHOP Webseite online, bevor überhaupt Code online war
    • django CMS wird u.a. mit djee.se für alle austestbar gemacht
    • divio AG als Partner für python.org Relaunch
